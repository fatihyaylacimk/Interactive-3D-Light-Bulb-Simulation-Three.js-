# Interactive 3D Light Bulb Simulation

This project is an interactive 3D light bulb demo built using JavaScript and Three.js.

The application simulates a realistic light bulb that can be turned on and off by pulling a rope or clicking the switch. The project focuses on visual effects, UI interaction, and WebGL-based 3D rendering.

---

## Features

Interactive rope pull mechanism  
Light bulb ON/OFF toggle system  
Real-time glow lighting effect  
Particle animation when the bulb turns on  
Energy usage counter  
Code UI panel simulation  
Glass breaking animation after multiple interactions  
Smooth camera and lighting effects

---

## Interaction

User can interact with the light bulb in two ways:

Pull the rope with the mouse  
Click on the switch

Each interaction toggles the light state.

If the bulb is toggled multiple times, the glass will break and the bulb stops working.
